# Biblioteca Escolar

Sistema simples de gestão da biblioteca da escola, feito para ser fácil de entender e usar em sala de aula.

## O que o sistema faz (MVP)
- Cadastro e login com matrícula (sem senha complicada no início)
- Lista de livros com busca rápida por título ou autor
- Registrar quando alguém pega ou devolve um livro
- Mostrar quem está com o livro no momento
- Interface que funciona bem no tablet ou celular da escola

## Nosso STACK
- **Backend** → Python + FastAPI (rápido e fácil de aprender)
- **Frontend** → React + Tailwind CSS (telas bonitas e responsivas)
- **Banco de dados** → PostgreSQL (produção) ou SQLite (para testar no computador de casa/escola)
