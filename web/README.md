# 🎨 EGM Lib - Frontend (Web)

Este é o dashboard e a interface do aluno do sistema de biblioteca, focado em ser rápido, responsivo e fácil de usar.

## 🛠️ Tecnologias Utilizadas
* **React.js**: Biblioteca base.
* **Vite**: Build tool ultra rápida.
* **Tailwind CSS**: Estilização via classes utilitárias.
* **Lucide React**: Biblioteca de ícones.
* **Axios**: Para chamadas à API do Backend.

## 🚀 Como rodar o ambiente de desenvolvimento

1. Certifique-se de ter o **Node.js (v18+)** instalado.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor local:
   ```bash
   npm run dev
   ```
4. Acesse no navegador: `http://localhost:5173`

## 📂 Estrutura de Pastas
* `src/components/`: Componentes reutilizáveis (Botões, Inputs, Cards).
* `src/pages/`: Telas completas (Login, Home, Dashboard).
* `src/assets/`: Imagens e estilos globais.
* `src/services/`: Configurações do Axios para conectar no Backend.

---
**⚠️ Atenção:** Antes de subir qualquer alteração, verifique se o comando `npm run build` passa sem erros.
