# Frontend - EGM Lib (SPA)

## Tech Stack
* React 18 (Vite)
* Tailwind CSS
* Lucide React
* Axios

## Scripts
- npm install: Deps
- npm run dev: Dev Server
- npm run build: Build/Dist

## Estrutura
- src/components: UI Components
- src/pages: Views
- src/services: API/Axios
- src/hooks: Custom Logic
